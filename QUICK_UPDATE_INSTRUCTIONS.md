# Инструкция по быстрому обновлению сервера

Приложение зависало на этапе "Загрузка..." из-за ожидания ответа от API.
Я добавил таймауты (10-15 секунд) и fallback логику (заглушки), чтобы приложение загружалось даже если сервер отвечает медленно.

Для применения исправлений выполните следующие действия на сервере (VPS):

### 1. Подключитесь к серверу (если есть доступ)
```bash
ssh root@185.166.197.49
# Пароль: qXY.W3,,Be?@gb
```
*(Если SSH не работает, используйте консоль в панели управления хостингом)*

### 2. Запустите скрипт обновления
Вставьте и выполните эту команду (одной строкой):

```bash
cd /var/www/spor3s-app && git pull origin main && cd spor3s-app && npm install && npm run build && pm2 restart all
```

### Что делает эта команда:
1.  Обновляет код из GitHub (`git pull`).
2.  Пересобирает приложение Next.js с новыми исправлениями.
3.  Перезапускает процессы (`pm2 restart all`).

---

Если у вас **нет доступа к консоли**, но настроен автоматический деплой через GitHub Actions:
Изменения уже отправлены в репозиторий. Если GitHub Actions настроен корректно, деплой начнется автоматически. 
Проверьте вкладку "Actions" в репозитории GitHub.
